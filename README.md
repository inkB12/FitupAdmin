# FITUP Admin

Admin dashboard template for FITUP, built with Next.js App Router + TypeScript + Tailwind CSS.

## Included pages

- `/login`
- `/dashboard`
- `/users`
- `/trainers`
- `/packages`
- `/blogs`
- `/transactions`
- `/settings`

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes

- Current project uses mocked data from `lib/admin-data.ts`.
- UI is ready for integrating authentication and backend APIs.
- Brand style and color direction match FITUP public website.
